Detailed Description

FontPalette holds set of fonts. It should be used within a Theme to define its font set:

import QtQuick 2.4
import quey.ui 0.1

Theme {
    FontLoader {
        id: myFont
        source: "myfont.ttf"
    }

    font: FontPalette {
        regular: Qt.font({
            family: myFont.name,
            pixelSize: 16
        })
    }
}

It is recommended to use FontLoader to load custom fonts.
