Detailed Description

It is recommended to use ApplicationView as the root component for any Quey UI based application.

ApplicationView provides layer components neccessary to display popups like Dialog or Menu. Those layers have a z value of 25. Please keep the z values of your items under this value unless you want those items to be displayed above popups.

import QtQuick 2.4
import quey.ui 0.1

ApplicationView {
    width: Units.dp(640)
    height: Units.dp(480)

    title: "My Awesome Application"

    Button {
        anchors.centerIn: parent
        text: "Hello World"
    }
}
